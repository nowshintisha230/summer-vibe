# 🌴 Summer Vibe

## 📌 Project Purpose

Summer Vibe is a modern web application designed to showcase and explore summer-related products like sunglasses, beach accessories, skincare items, and more. The goal is to provide users with a visually appealing and smooth shopping-like experience.

---

## 🚀 Live URL

https://summer-vibe.vercel.app

---

## ✨ Key Features

✔ Product Listing Page – Displays all summer products in a responsive grid layout
✔ Product Details Page – View detailed information about each product
✔ User Profile System – Users can view and update their profile information
✔ Fast Performance – Built with Next.js for optimized speed
✔ Fully Responsive – Works smoothly on mobile, tablet, and desktop
✔ Modern UI Design – Clean and attractive design using Tailwind CSS
✔ Dynamic Routing – Each product has its own unique page
✔ Optimized Images – Uses Next.js Image component for better performance

---

## 📦 NPM Packages Used

✔ next – React framework for production
✔ react – UI library
✔ react-dom – DOM rendering
✔ tailwindcss – Utility-first CSS framework
✔ @heroui/react – UI component library
✔ lucide-react – Icon library
✔ next/image – Image optimization (built-in)
